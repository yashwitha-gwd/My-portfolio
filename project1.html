Student Performance Early Warning System(Risk Level Prediction)

Risk Level Prediction System using Machine Learning. This project demonstrates how a trained ML model can be integrated with a web application and deployed as a live interactive tool.

Problem Statement

Predicting student performance using their mark.predicting risk level which gives warning !.

Project Overview

The system takes user input through a web interface and predicts the corresponding risk level using a trained Machine Learning model. This project helped me understand the complete lifecycle of a Machine Learning application — from model development to real-time deployment.

Tools & Technologies


* Python
* Scikit-learn for Machine Learning
* Flask for backend web framework
* HTML & CSS for user interface
* Pickle for saving the trained model
* GitHub for version control
* Render

Results

This project gave me hands-on experience in Machine Learning, backend development, and deploying real-world AI applications.
